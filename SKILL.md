# DuckDuckGo Search

## Instructions

Call the `run_js` tool using `index.html` and a JSON string for `data` with the following fields:
- **query**: Required. คำค้นหาที่ต้องการเสิร์ชบนอินเตอร์เน็ต เช่น วันนี้มีบอลโลกคู่ไหนเตะบ้าง
